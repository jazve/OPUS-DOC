# GitHub Topics 建议

为了提高项目的可发现性，建议在GitHub仓库设置中添加以下topics：

## 核心技术标签
- `ai`
- `agent` 
- `chatgpt`
- `claude`
- `gemini`
- `prompt-engineering`
- `ai-framework`
- `artificial-intelligence`

## 功能特性标签
- `no-code`
- `automation` 
- `rag`
- `mcp`
- `llm`
- `nlp`
- `conversational-ai`
- `ai-assistant`

## 应用场景标签
- `workflow`
- `productivity`
- `business-automation`
- `ai-tools`

## 中文标签（如果适用）
- `智能体`
- `人工智能`
- `提示词工程`

## 如何添加topics
1. 进入GitHub仓库页面
2. 点击右侧的设置齿轮图标（在About部分）
3. 在Topics字段中添加上述标签
4. 保存设置

建议选择10-20个最相关的topics，避免标签过多。
